IO的深入学习 ​

概述 ​

在很久之前写C语言作业时遇到一个问题, 从stdin获取用户输入时会获取到两次用户输入, 以下是一个小demo

c
#include <stdio.h>

void print_menu()
{
    printf("1.abc\n2.bcd\n3.ccc\n>>>");
}

void process(char cho)
{
    switch (cho)
    {
    case '1':
        printf("abc\n");
        break;
    case '2':
        printf("bcd\n");
        break;
    case '3':
        printf("ccc\n");
        break;
    default:
        printf("err\n");
        break;
    }
}

int main()
{
    while (1)
    {
        print_menu();
        char choic = getchar();
        printf("\n");
        process(choic);
    }
}
